Transparencia técnica

Este es el motor. Horas reducidas a segundos.

Un agente de generación de evaluaciones FPE que extrae el temario, aplica criterio pedagógico y devuelve JSON estructurado listo para exportar al LMS.

agente_fpe.py
Python · OpenAI SDK 
import openai
import json
from document_parser import extract_text

def generar_evaluacion_fpe(ruta_temario, modulo):
    # 1. Extracción — ingesta del manual en bruto
    contenido = extract_text(ruta_temario)

    # 2. Prompting con criterio pedagógico y taxonomía Bloom
    instrucciones = f"""
    Eres un experto en diseño curricular FPE.
    Basándote en el contenido, genera 10 preguntas de opción múltiple
    para evaluar el módulo: {modulo}.
    Aplica taxonomía de Bloom (nivel aplicación y análisis).
    Devuelve estrictamente un JSON válido con la estructura:
    {{ "preguntas": [ {{ "enunciado": ..., "opciones": [...], "correcta": 0 }} ] }}
    """

    # 3. Llamada al LLM con temperatura baja para máxima precisión
    respuesta = openai.ChatCompletion.create(
        model="gpt-4-turbo",
        messages=[
            {"role": "system", "content": instrucciones},
            {"role": "user",   "content": contenido}
        ],
        temperature=0.2  # Baja temperatura → alta consistencia
    )

    # 4. Parseo y validación antes de entregar al docente
    resultado = json.loads(respuesta.choices[0].message.content)
    return resultado  # → Listo para exportar a Moodle/Canvas
output — 10 preguntas generadas en 4.3s
✓ Módulo: Fundamentos de Redes · UF1 | ✓ Taxonomía Bloom: Aplicación + Análisis | ✓ JSON válido exportable a LMS
